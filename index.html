Linguagem Java Script

Caracteristicas

  • Linguagem de Programação
  • Linguagem Interpretada
  • Originalmente foi criado como uma linguagem front-end, ou seja, ele era interpretado e executado no cliente, navegador
  • Atualmente ele tambem pode ser executado no Back-End através do NodeJS
  • É case-sesitive
  • Fracamente tipada, pode ser tipada se programa com TypScript
  • Sintaxe similar a Linguagem C, Java
  • Estruturada, Orientada a objetos
